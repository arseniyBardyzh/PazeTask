# PazeTask

## Description
This is implementation createPayment POST request from [Paze API](https://app.paze.eu/docs/#tag/Payment/operation/createPayment)

## Tech Stack
* Java17
* SpringBoot 2.6.14

## Getting Started
You can run the application by maven repository 
    
    maven clean install
